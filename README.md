# QuickMenu

A minimalistic **QuickShell widget** for **Hyprland**, showing essential system stats and quick controls — built with a **dark, sharp-edged design**.

---

## ✨ Features

- 📊 **System Monitor**
  - CPU usage
  - RAM usage
  - Disk usage
- 🔊 **Volume Slider**
- 🎵 **Media Player Controls**
- ⚙️ **Quick Toggles**
  - Bluetooth
  - VPN
  - Hyprlock

---

## 🧩 Requirements

Make sure you have:

- **Hyprland**
- **QuickShell**

Install QuickShell on Arch-based systems:

```bash
yay -S quickshel
```
